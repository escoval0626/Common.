# McAfee Security Evaluation Request Guide

マカフィーの「安全かどうか不明です」警告を解除するための申請手順です。
以下のポータルサイトから申請を行ってください。

## 1. 申請ポータルへのアクセス
**URL**: [https://sitelookup.mcafee.com/](https://sitelookup.mcafee.com/)

---

## 2. アカウント登録（初回のみ）
1. 右上の **[Register]** からアカウントを作成してください。
2. 登録したメールアドレスに確認メールが届くので、本文のリンクをクリックして有効化します。
3. 有効化後、ログインします。

---

## 3. 申請内容の設定
ログイン後、以下の項目を入力・選択してください。

- **Choose Customer type**: `McAfee SiteAdvisor/WebControl (Enterprise)` を選択
- **URL**: `common-ordinarydays.com` を入力
- **[Check URL]** をクリック

---

## 4. 再評価のリクエスト (Submit for Review)
現在の評価が「Unrated（未評価）」と表示されますので、以下の情報を入力して **[Submit URL for Review]** をクリックします。

### カテゴリの選択 (Check up to 3 Categories)
- `Business`
- `Professional Services`
- `General News` (または `Marketing/Merchandising`)

### コメント欄 (Optional comment)
※以下の英文をコピー＆ペーストしてください。

> **English (Recommended)**:
> This is a newly established official portfolio website for "Common. Studio," a professional creative studio based in Nagareyama, Japan. The website only contains professional design works and photographs, with no malicious links or harmful content. We have properly implemented SSL and SEO best practices. Please review and update the safety rating to "Safe."
>
> **日本語訳 (内容確認用)**:
> これは千葉県流山市を拠点とするクリエイティブスタジオ「Common. Studio」の新しい公式サイト兼ポートフォリオです。プロフェッショナルなデザイン実績と写真のみを掲載しており、悪意のあるリンクや有害なコンテンツは一切含まれていません。SSL化も適切に実施済みです。安全性の評価を「安全」に更新いただけますようお願いいたします。

---

## 5. 完了後の流れ
- 申請後、チケットIDが発行されます。
- 通常 **3〜5営業日** 程度で評価が「安全（緑色のチェック）」に更新されます。
- 更新されると、Google等の検索結果に表示されるマカフィーの警告ポップアップも自動的に消滅します。
